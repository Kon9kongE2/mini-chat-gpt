# Dataset

- File: `lucky_day.txt`
- Text: 현진건, 「운수 좋은 날」, 1924
- Source: Korean Wikisource  
  https://ko.wikisource.org/wiki/운수_좋은_날

## Preprocessing

줄바꿈, 한자 병기, 장식 문자, 말줄임표, 대시 기호를 제거하거나 정리하였다.  
마침표, 쉼표, 따옴표, 괄호 등 기본 문장부호는 유지하였다.

## Purpose

`notebook_06.ipynb`에서 한국어 텍스트를 character-level GPT에 적용해 보기 위한 학습 데이터이다.
